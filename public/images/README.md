# ジャンルのスワイプカード写真

このフォルダに各ジャンルの写真を置くと、スワイプカードが全面写真表示になります。
ファイルが無いジャンルは、これまで通り絵文字表示にフォールバックします。

- 形式：JPEG推奨（`.jpg`）。縦長（portrait）だとカードにきれいに収まります
- ライセンス：商用可・帰属不要のフリー素材を推奨
- ファイル名は下記のとおり（ジャンルIDに対応）：

| ファイル名 | ジャンル |
|---|---|
| ramen.jpg | ラーメン |
| sushi.jpg | 寿司 |
| yakiniku.jpg | 焼肉 |
| korean.jpg | 韓国料理 |
| italian.jpg | イタリアン |
| french.jpg | フレンチ |
| chinese.jpg | 中華 |
| curry.jpg | カレー |
| soba.jpg | そば・うどん |
| izakaya.jpg | 居酒屋 |
| ethnic.jpg | タイ・エスニック |
| cafe.jpg | カフェ飯 |
